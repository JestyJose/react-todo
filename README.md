# React To-Do List

A simple and interactive To-Do List application built using React.js to help manage daily tasks easily.

## Features
- Add tasks using the input box and Add button
- Click on a task to mark it as completed
- Delete tasks using the ❌ button

## How to Use
1. Enter a task in the input box.
2. Click **Add** to add the task to the list.
3. Click on a task to mark it as completed.
4. Click ❌ to remove a task from the list.

   The website can be accessed from: https://jestyjose.github.io/react-todo/
